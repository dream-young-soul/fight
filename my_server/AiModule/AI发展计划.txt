。事实库目前只有独立事件和多重事件两种，以后可考虑允许指定事实库中有多少条同类事件。
。目前是通过EVENT和ACTION实现低层逻辑，以后可细化上层逻辑，减少EVENT和ACTION的种类。

。词法规则：
	。FACT		:=  [ '!' ]  <标识符>  [ '<'  <参数>  {  ','  <参数>  }  '>'  ]   [ '('  <参数>  {  ','  <参数>  }  ')' ]
	。标识符	:=  <字母>  {  <chars>  }
	。参数		:=  <符号> | <字符串> | <数值>
	。符号		:=  '_' | <标识符>
	。<字符串>	:=  "..."
	。<数值>	:=  [ '-' ]  <数字> { <数字> }  [ <chars> ]
	。chars		:=  <字母> | <数字> | '_'
	。FACT_SET	:=  <FACT>  { ','  <FACT> }
